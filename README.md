# VSB2DB
VRMSpringBone to DynamicBone Converter

### 必要なもの

- Unity (2018.4.9にて動作確認済み)
- UniVRM (最新版推奨　v0.53動作確認済み)
- DynamicBone (AssetStoreで購入してください)

### 使い方
1.UnityでProjectを作成します。

2.VSB2DBとUniVRM、DynamicBoneをimportします。

3.VRMSpringBoneが設定されたモデルを読み込みます。

4.上のメニューバーからVRM→VSB2DBを選択肢してウィンドウを開きます。

5.Taget Modelに変換するモデルをセットします。

6.Start VSB2DBを押します。

7.一瞬でConvertが終わります。
